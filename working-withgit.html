Working with Git

1. Add files to your project

This can be done via the touch command or through the interface as usual

Example: $touch index.html && touch main.css

2. Add new or modified files to staging

Once we add new files or modify existing files we can tell git to track them by adding them to staging for the nect commit(think of commits as checkpoints). This will let git know what files the next checkpoint will consist of.

$ git add

Or

$ git add .

Your files are now on stage to be included in the next commit/checkpoint
