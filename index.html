Starting a new project

1. Creating a new folder for your project

You can use the command $ mkdir(folder-name)or you can do this through interface

2. Initialize Git Repositort inside PRoject

Navigate to the project folder inside of the terminal and run the following command
$git init

You should see the following output:

Iniitialized empty Git repository in User/(ParentFolder)/(project)/.git/

3. Open a browser and go to Github and create a new repository

Once you create a new repo, you will copy and paste the following commands into terminal. These commands set the git remote to point to your project folder...

$ git remote add origin https://github.com/mlaw-nycda/day5.git

You're all set!

Now read about adding files to your project and pushing your code to Github
